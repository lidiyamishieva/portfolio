# Week 4 - Exercise 2

The files in this directory contain the reproducible research compendium for an example simulation study by Boulesteix et al. (2020).

Link to the paper: <https://bmjopen.bmj.com/content/10/12/e039921>

For reproducing the manuscript download the directory, open *exercise2.Rproj* and within this project run *reproducible-manuscript-simulation-study.qmd*.
